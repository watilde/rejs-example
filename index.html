    rejs-example

    Code examples using rejs. Test and buid code are using npm run-scripts.

    How to use is as follows:

    Build with cli

    $ npm run cli
$ npm test

    Build with grunt

    $ npm run grunt
$ npm test

    Build with gulp

    $ npm run gulp
$ npm test
